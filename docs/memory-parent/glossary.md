# Glossaire Altio

| Terme | Signification |
|-------|---------------|
| Altio | Nom de la plateforme/marketplace |
| Owner | Propriétaire de logement saisonnier (client donneur d'ordre) |
| Provider | Prestataire de services (ménage, conciergerie, maintenance) |
| Mission | Tâche assignée à un provider pour un owner (ménage, check-in, etc.) |
| Property | Logement/bien immobilier enregistré par un owner |
| RLS | Row Level Security (Supabase) |
| Edge Function | Fonction serverless Supabase (runtime Deno) |
| EAS | Expo Application Services |
| Factur-X | Format e-facture français (PDF + XML CII) |
| SIREN | Identifiant entreprise France (9 chiffres) |
| TVA intracom | Numéro de TVA intracommunautaire (FR + 11 chars) |
| CGU | Conditions Générales d'Utilisation |
| CGV | Conditions Générales de Vente |
| iCal | Format calendrier .ics (Airbnb, Booking.com) |
| Stripe Connect | Compte Stripe pour les prestataires (virements) |
| OTA Update | Over The Air — mise à jour sans re-soumission stores |
| code-checker | Skill Claude Code pour vérifier les corrections |
